# Blackjack game 


This project implements the game Blackjack. To do this, the website was created using the HTML and CSS with Bootstrap. The implementation was done using object-oriented JavaScript.

It also has a debug mode. To activate it, simply go to the HTML file and uncomment the lines between 125 and 128.
